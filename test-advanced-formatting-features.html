Feature Testing Instructions

  1. Lists: Use the bullet and numbered list buttons in the toolbar, then use indent/outdent buttons to create nested lists
  2. Images: Click the image button to upload a local image file, then click on inserted images to resize them
  3. Tables: Click the table button to open the table insertion modal, hover over cells to select size
  4. Colors: Click the text color or highlight color buttons to open color palettes
  5. Formatting: Select text and use the formatting buttons to apply bold, italic, underline
  6. Headings: Use the heading dropdown to apply different heading levels
